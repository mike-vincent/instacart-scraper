instacart-order-history-download

A command-line tool to download your Instacart order history.

1. Install

pip install instacart-order-history-download

2. Find Your Session ID

Log into instacart.com, open DevTools → Cookies, and copy _instacart_session_id.

3. Download Your Receipts

instacart-order-history-download \
  --instacart-session-id "YOUR_SESSION_ID"

Build from Source

git clone https://github.com/mike-vincent/instacart-order-history-download.git
cd instacart-order-history-download
python3 instacart_order_history_download.py --help

Homebrew

brew install mike-vincent/tap/instacart-order-history-download

MacPorts

sudo port install instacart-order-history-download
