instacart-order-history-fetch

Download your Instacart receipts as JSON, CSV, or text.

Homebrew

brew tap mike-vincent/instacart-order-history-fetch
brew install instacart-order-history-fetch

Instacart Session ID

Log into instacart.com, open your browser DevTools, go to Application → Cookies, and copy the value of _instacart_session_id. It looks like this:

WlhsS05XTXpTVzFrVjFaNlRETkNkR0ZKUW5OaFYwNXNZM2xCYVZwWVNq...
instacart-order-history-fetch \
  --instacart-session-id "WlhsS05XTXpTVzFr..."

Usage

# Fetch all orders as JSON
instacart-order-history-fetch \
  --instacart-session-id "abc123..."

# Fetch last 6 months to CSV
instacart-order-history-fetch \
  --instacart-session-id "abc123..." \
  --months 6 -f csv

# Fetch to file
instacart-order-history-fetch \
  --instacart-session-id "abc123..." \
  -o orders.json
