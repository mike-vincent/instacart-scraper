instacart-order-history-download

A command-line tool to download your Instacart order history.

## Download Your Instacart Receipts

instacart-order-history-download \
  --instacart-session-id "WlhsS05XTXpTVzFrVjFaNlRETkNkR0ZKUW5OaFYwNXNZM2xCYVZwWVNqLS1LUlZ5UHVaUXVkbzZ2dEpsU29qU2ZnPT0tLTQ4MTY1YmM1N2ZhYmUwZWU1MGYxZDAxNjIxMTM2ZGExOTZmOGYyYTE="

## Find Your Instacart Session ID

Log into instacart.com, open your browser DevTools, go to Cookies, and copy the value of _instacart_session_id.

## Homebrew

brew tap mike-vincent/instacart-order-history-download
brew install instacart-order-history-download

## MacPorts

sudo port install instacart-order-history-download

## Build from source

git clone https://github.com/mike-vincent/instacart-order-history-download.git
cd instacart-order-history-download
chmod +x instacart-order-history-download
